# FEATURES

* Renders MakerJS Models Inside VS Code
* Saves Generated SVG Alongside JS Source
* Update Preview Window Automatically On Save

# HOWTO

* Right Click MakerJS Text File Header
* Select MAKERJS_WATCH From Context Menu
